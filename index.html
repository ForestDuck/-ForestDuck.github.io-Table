CSS FlexBox

CSS FlexBox Tags
Tag Comands Description
display flex | inline-flex This defines a flex container; inline or block depending on the given value. It enables a flex context for all its direct children.
flex-direction row | reverse-row | column | reverse-column Defining the direction flex items are placed in the flex container.
flex-wrap nowrap | wrap | wrap-reverse By default, flex items will all try to fit onto one line. You can change that and allow the items to wrap as needed with this property.
flex-flow row | reverse-row | column | reverse-column / nowrap | wrap | wrap-reverse This is a shorthand for the flex-direction and flex-wrap properties, which together define the flex container’s main and cross axes. The default value is row nowrap.
flex-grow numbers ( 1-4) This defines the ability for a flex item to grow if necessary. It accepts a unitless value that serves as a proportion. It dictates what amount of the available space inside the flex container the item should take up.
flex-shrink numbers ( 1-4) This defines the ability for a flex item to shrink if necessary.
flex-basis 0 | auto; This defines the default size of an element before the remaining space is distributed.
justify-content flex-start | flex-end | center | space-between | space-around | space-evenly | start | end | left | right; This defines the alignment along the main axis. It helps distribute extra free space leftover when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size. It also exerts some control over the alignment of items when they overflow the line.
align-items stretch | flex-start | flex-end | center | baseline | first baseline | last baseline | start | end | self-start | self-end; This defines the default behavior for how flex items are laid out along the cross axis on the current line. Think of it as the justify-content version for the cross-axis (perpendicular to the main-axis).
align-content flex-start | flex-end | center | space-between | space-around | space-evenly | stretch | start | end | baseline | first baseline | last baseline; This aligns a flex container’s lines within when there is extra space in the cross-axis, similar to how justify-content aligns individual items within the main-axis.
gap, row-gap, column-gap gap, row-gap, column-gap The gap property explicitly controls the space between flex items. It applies that spacing only between items not on the outer edges.
